# Polixir_Fapiao - 打车发票自动化处理工具

这是一个用于自动处理打车PDF发票（Fapiao）的工具，旨在将繁琐的手动操作流程自动化，提高报销效率。

## 主要功能

- **自动合并**：将指定文件夹内的PDF文件两两配对合并。
- **智能重命名**：根据PDF内容中的“行程起止日期”自动对合并后的文件进行重命名。
- **信息提取**：从PDF中精确提取关键信息，包括行程日期、发票号码和合计金额。
- **生成报表**：将所有提取的信息自动汇总，生成一个格式化的Excel（`.xlsx`）报销单。

---

## 项目结构

```
Polixir_Fapiao/
├── download_pdf/           # 将下载的发票和行程单放到这里，如果是发票则命名为1,3,5...pdf，如果是行程单则命名为2,4,6...pdf。需要注意的是发票和行程单需要一一对应，如1.pdf和2.pdf为一个完整行程的发票和行程单，3.pdf和4.pdf为另一个完整行程的发票和行程单，以此类推。
├── merge_pdf/                  # 存放发票和行程单合并后的PDF文件，即一个完整行程的报销凭证。
├── 报销单.xlsx             # (自动生成) 最终的Excel报销文件
├── merge_pdfs.py           # <--- 合并PDF的脚本
├── rename_pdfs.py          # <--- 重命名PDF，为2010-01-01_打车.pdf
├── create_expense_report.py# <--- 生成最终的Excel报表
├── clean_project.py        # (可选) 运行此脚本来清理所有生成的文件
├── requirements.txt        # 项目依赖库
└── README.md               # 项目说明文件
```

## 如何使用

### 1. 环境安装

首先，你需要安装项目所需的Python库。在项目根目录下打开终端，运行以下命令：

```bash
pip install -r requirements.txt
```

### 2. 放置文件

将所有需要处理的原始PDF发票文件放入 `init_dir` 文件夹。

### 3. 运行处理脚本


```bash
python create_expense_report.py
```

脚本运行完毕后，你会在 `merge` 文件夹中找到重命名后的PDF，并在根目录下找到 `报销单.xlsx` 文件。

### 4. (可选) 清理项目

当你需要处理新的一批发票时，可以运行清理脚本来删除所有之前生成的文件，让项目恢复到初始状态：

```bash
python3 clean_project.py
```
