# Polixir_Fapiao - 发票自动化处理 Web 应用

这是一个用于自动处理PDF发票（Fapiao）的Web应用，旨在将繁琐的手动操作流程在线化，提高报销效率。

## 主要功能

- **Web 界面**：通过浏览器上传 PDF 文件，操作简单直观。
- **自动合并**：将上传的 PDF 文件两两配对合并。
- **智能重命名**：根据 PDF 内容中的“行程起止日期”自动对合并后的文件进行重命名。
- **信息提取**：从 PDF 中精确提取关键信息，包括行程日期、发票号码和合计金额。
- **生成报表**：将所有提取的信息自动汇总，生成一个格式化的 Excel（`.xlsx`）报销单，并提供下载。

## 如何使用

### 1. 环境安装

首先，你需要安装项目所需的Python库。在项目根目录下打开终端，运行以下命令：

```bash
pip install -r requirements.txt
```

### 2. 运行 Web 应用

在项目根目录下运行以下命令来启动 Flask Web 服务器：

```bash
python app.py
```

服务器启动后，你会在终端看到类似以下的输出：

```
 * Running on http://127.0.0.1:5001/
```

### 3. 打开浏览器并使用

在你的浏览器中打开 `http://127.0.0.1:5001/`。

1.  点击“选择 PDF 文件...”按钮，选择一个或多个发票 PDF 文件。
2.  点击“上传并生成报销单”按钮。
3.  处理完成后，页面会显示一个成功消息和一个下载链接。
4.  点击下载链接，即可获取生成的 `报销单.xlsx` 文件。
